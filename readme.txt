Elite-Board v2.3.180220

Внимание! Это не кумулятивное обновление! Только для обновления сборки 2.3.171107!

Узнать номер текущей сборки можно обратившись на своём сайте по адресу /version.htm
При установке обновлений строго соблюдайте их последовательность!

Список изменений:

- добавлена возможность добавления стоимости пакета для пакетов услуг в админке;
- переписан интерфейс заказа и оплаты, уменьшено количество действий для оплаты;
- пользователь с ненулевым балансом теперь оплачивает только недостающую часть суммы заказа, остальное списывается с баланса;
- исправлена ошибка в админке - отображение магазинов, у которых был удалён пользователь-владелец;
- устранено зависание индексирования при использовании редиректов в пользовательской части сайта;
- исправлена неправильная ссылка в футере шаблона flame;
- исправлена ошибка с пустым именем автора и email после редактирования объявления от незарегистрированного пользователя;
- исправлена ошибка при сортировке рубрик и регионов;
- исправлена проблема с периодической нагрузкой на windows серверах;
- исправлена ошибка с неправильным отображением рубрик объявлений на главной странице;
- исправлена ошибка с неправильным размещением блоков;
- исправлена ошибка при сохранении прав пользователя.


Порядок установки:

- замените файлы на сервере соответствующими файлами из архива
